1. Giới thiệu về Socket

Socket là giao diện lập trình mạng cho phép hai tiến trình (process) giao tiếp qua mạng.
Socket hoạt động như một điểm cuối (endpoint) của kênh truyền dữ liệu giữa client và server.

Có hai loại socket phổ biến:

  • TCP (Transmission Control Protocol): Đảm bảo truyền dữ liệu tin cậy, có kiểm tra lỗi.
  • UDP (User Datagram Protocol): Truyền dữ liệu nhanh hơn nhưng không đảm bảo.

2. Socket trong Java

Tạo Server TCP

import java.io.*;
import java.net.*;

public class Server {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(1234);
        System.out.println("Server đang chờ kết nối...");
        Socket socket = server.accept();
        System.out.println("Client đã kết nối!");
        
        BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
        PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

        String message = in.readLine();
        System.out.println("Client: " + message);
        out.println("Xin chào từ Server!");
        
        socket.close();
        server.close();
    }
}

Tạo Client TCP

import java.io.*;
import java.net.*;

public class Client {
    public static void main(String[] args) throws IOException {
        Socket socket = new Socket("localhost", 1234);
        
        BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
        PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
        
        out.println("Hello Server!");
        System.out.println("Server: " + in.readLine());
        
        socket.close();
    }
}

3. Socket trong Python

Tạo Server TCP

import socket

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.bind(('localhost', 1234))
server.listen(1)

print("Server đang chờ kết nối...")
conn, addr = server.accept()
print(f"Kết nối từ {addr}")

data = conn.recv(1024).decode()
print("Client:", data)
conn.sendall("Xin chào từ Server!".encode())

conn.close()
server.close()

Tạo Client TCP

import socket

client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client.connect(('localhost', 1234))

client.sendall("Hello Server!".encode())
print("Server:", client.recv(1024).decode())

client.close()

4. Socket trong C#

Server TCP

using System;
using System.Net;
using System.Net.Sockets;
using System.Text;

class Server {
    static void Main() {
        TcpListener server = new TcpListener(IPAddress.Any, 1234);
        server.Start();
        Console.WriteLine("Server đang chờ kết nối...");

        TcpClient client = server.AcceptTcpClient();
        NetworkStream stream = client.GetStream();

        byte[] buffer = new byte[1024];
        int bytesRead = stream.Read(buffer, 0, buffer.Length);
        string message = Encoding.UTF8.GetString(buffer, 0, bytesRead);
        Console.WriteLine("Client: " + message);

        byte[] reply = Encoding.UTF8.GetBytes("Xin chào từ Server!");
        stream.Write(reply, 0, reply.Length);

        client.Close();
        server.Stop();
    }
}

Client TCP

using System;
using System.Net.Sockets;
using System.Text;

class Client {
    static void Main() {
        TcpClient client = new TcpClient("localhost", 1234);
        NetworkStream stream = client.GetStream();

        byte[] data = Encoding.UTF8.GetBytes("Hello Server!");
        stream.Write(data, 0, data.Length);

        byte[] buffer = new byte[1024];
        int bytesRead = stream.Read(buffer, 0, buffer.Length);
        Console.WriteLine("Server: " + Encoding.UTF8.GetString(buffer, 0, bytesRead));

        client.Close();
    }
}

5. Tổng kết

Ngôn ngữ Lớp chính Ghi chú
Java Socket, ServerSocket Cần đóng socket sau khi dùng
Python socket.socket Dễ dùng, hỗ trợ cả TCP/UDP
C# TcpClient, TcpListener Có hỗ trợ bất đồng bộ (async/await)

Socket là nền tảng của các ứng dụng mạng như chat, game online, IoT.
Hiểu được socket là bước đầu để xây dựng các hệ thống client-server hiệu quả.
