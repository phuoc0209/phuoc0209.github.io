Giới thiệu ngôn ngữ Java

Đăng ngày 20/10/2025 lúc 00:00

Java là một ngôn ngữ lập trình hướng đối tượng (OOP) mạnh mẽ, được phát triển bởi Sun Microsystems (nay thuộc Oracle) vào năm 1995.
Đây là ngôn ngữ lập trình có khẩu hiệu nổi tiếng:

“Write once, run anywhere” – Viết một lần, chạy ở mọi nơi.

1. Giới thiệu chung

Java được thiết kế để chạy trên máy ảo Java (JVM), giúp chương trình có thể hoạt động trên mọi hệ điều hành (Windows, macOS, Linux…).
Điều này có nghĩa là chỉ cần bạn biên dịch chương trình Java một lần, bạn có thể chạy nó trên bất kỳ nền tảng nào có cài Java Runtime Environment (JRE).

Một số đặc điểm nổi bật của Java:

  • ✅ Đa nền tảng (cross-platform)
  • ✅ Hướng đối tượng (Object-Oriented)
  • ✅ Quản lý bộ nhớ tự động (Garbage Collection)
  • ✅ Có thư viện phong phú và cộng đồng lớn
  • ✅ Dễ học, dễ sử dụng cho người mới bắt đầu

2. Cấu trúc cơ bản của chương trình Java

Mọi chương trình Java đều bắt đầu từ phương thức main().
Ví dụ:

public class Hello {
  public static void main(String[] args) {
    System.out.println("Hello Java!");
  }
}

Giải thích:

Thành phần Ý nghĩa
public Là phạm vi truy cập, cho phép chương trình chạy từ bên ngoài lớp.
class Hello Khai báo một lớp tên là Hello. Trong Java, mọi thứ đều nằm trong lớp.
public static void main(String[] args) Là phương thức chính (main), nơi chương trình bắt đầu chạy.
System.out.println("Hello Java!"); Lệnh in ra màn hình chuỗi "Hello Java!".

3. Biến và Kiểu dữ liệu trong Java

Java có 2 loại kiểu dữ liệu chính:

  1. Kiểu dữ liệu nguyên thủy (Primitive types)
  2. Kiểu đối tượng (Non-primitive / Reference types)

Ví dụ:

int age = 20;              // Số nguyên
double price = 19.99;      // Số thực
char grade = 'A';          // Ký tự
boolean isJavaFun = true;  // Giá trị logic
String name = "Tuấn";      // Chuỗi ký tự (đối tượng)

4. Cấu trúc điều khiển trong Java

➤ Cấu trúc if-else

int x = 10;

if (x > 5) {
  System.out.println("x lớn hơn 5");
} else {
  System.out.println("x không lớn hơn 5");
}

➤ Vòng lặp for

for (int i = 1; i <= 5; i++) {
  System.out.println("Số: " + i);
}

➤ Vòng lặp while

int count = 0;
while (count < 3) {
  System.out.println("Java thật tuyệt!");
  count++;
}

5. Lớp và Đối tượng (OOP cơ bản)

Java là ngôn ngữ hướng đối tượng, nghĩa là bạn mô tả thế giới thực qua lớp (class) và đối tượng (object).

Ví dụ:

public class Student {
  // Thuộc tính (biến)
  String name;
  int age;

  // Phương thức (hàm)
  void introduce() {
    System.out.println("Xin chào, mình là " + name + ", " + age + " tuổi.");
  }
}

public class Main {
  public static void main(String[] args) {
    Student s1 = new Student();   // Tạo đối tượng
    s1.name = "Tuấn";
    s1.age = 20;

    s1.introduce();
  }
}

Giải thích:

  • class Student: định nghĩa khuôn mẫu cho đối tượng “Sinh viên”.
  • name, age: thuộc tính (dữ liệu).
  • introduce(): phương thức (hành vi).
  • new Student(): tạo một đối tượng mới thuộc lớp Student.

6. Nhập và xuất dữ liệu

Java có lớp Scanner giúp người dùng nhập dữ liệu từ bàn phím.

import java.util.Scanner;

public class InputExample {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.print("Nhập tên của bạn: ");
    String name = sc.nextLine();
    System.out.println("Xin chào, " + name + "!");
  }
}

7. Một số khái niệm quan trọng khác

Khái niệm Mô tả ngắn gọn
Package Nhóm các lớp lại để quản lý dễ hơn.
Method Một hàm bên trong lớp.
Constructor Hàm đặc biệt được gọi khi tạo đối tượng.
Inheritance (Kế thừa) Cho phép lớp con kế thừa thuộc tính và phương thức của lớp cha.
Polymorphism (Đa hình) Cho phép một phương thức có thể hoạt động khác nhau tùy đối tượng.
Encapsulation (Đóng gói) Ẩn thông tin bên trong lớp, chỉ cung cấp những gì cần thiết.

8. Kết luận

Java là một ngôn ngữ bền vững, mạnh mẽ, được sử dụng trong:

  • Ứng dụng Android
  • Hệ thống doanh nghiệp
  • Trò chơi và phần mềm máy tính
  • Ứng dụng web và dịch vụ backend

🔸 Nếu bạn mới bắt đầu học lập trình, Java là lựa chọn tuyệt vời để rèn tư duy logic và nền tảng vững chắc.
